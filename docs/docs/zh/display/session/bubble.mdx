# Bubble 气泡

> 用于承载对话内容的气泡组件，支持 loading、透明背景、自定义加载动画等。

## 示例效果

<div className="playground-wrapper">
  <BubbleDemo />
</div>

```ts title="代码 BubbleDemo.tsx"
import { Bubble } from '@matechat/react';
import avatar from '../../public/avatar.png';

export default function BubbleDemo() {
  return (
    <div className="bubble-chat">
      <div className="bubble-row left">
        <img src={avatar} alt="Bot" className="avatar" />
        <Bubble text="您好，请问有什么可以帮您？" />
      </div>
      <div className="bubble-row right">
        <Bubble text="我想了解一下产品功能！" />
        <img src={avatar} alt="User" className="avatar" />
      </div>
      <div className="bubble-row left">
        <img src={avatar} alt="Bot" className="avatar" />
        <Bubble text="好的，请稍等..." isPending />
      </div>
    </div>
  );
}
```

## 参数

| 参数名       | 类型                                        | 默认值     | 描述                                                                  |
| ------------ | ------------------------------------------- | ---------- | --------------------------------------------------------------------- |
| `text`       | `string`                                   | `必填`  | 气泡中显示的文本，可包含 Markdown                                     |
| `background` | `transparent` \| `solid`               | `solid`  | 背景样式                                                              |
| `isPending`  | `boolean`                                  | `false`    | 是否显示加载动画                                                      |
| `pending`    | `React.ReactNode`                          | `默认动画` | 自定义加载内容                                                        |

> ✅ **提示**：还可接收原生 div props，如 `className`、`style` 等
